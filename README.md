# Panel Financiero en Python

## Descripción
Aplicación de consola desarrollada en Python que permite registrar ingresos y gastos, almacenarlos en archivos y calcular un balance financiero.

## Funcionalidades
- Registro de ingresos
- Registro de gastos
- Almacenamiento en archivos de texto
- Cálculo de balance total
- Manejo de errores básicos

## Tecnologías
- Python 3
- Manejo de archivos (txt)
- Estructuras básicas de programación
